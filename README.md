# Anchor Recycling Website

Static website for Anchor Recycling LLC.

## Files

- `index.html` contains the page content and structure.
- `styles.css` contains the responsive layout and visual design.
- `script.js` contains the mobile navigation and header scroll behavior.
- `assets/` contains site images and logo files.

## Local Preview

Open `index.html` in a browser to view the site locally.

## Contact Form

The current contact form uses a `mailto:` action pointed at `info@anchor-recycling.com`. This opens the visitor's email application instead of submitting through a server. For production use, connect the form to a hosted form service or backend endpoint.
